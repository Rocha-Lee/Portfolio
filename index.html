Conhecimentos

Aqui estão alguns dos conhecimentos que adequiri tentando criar jogos para o Unity

Blender

Para usar o Untiy foi preciso aprender um pouco de modelagem 3D, texturização e etc. o Blender é uma otima ferramenta pois ele é open source e consegue fazer tudo o que necessario para a criação de um jogo.

Unity

O Unity é um um motor de jogo criado pela Unity Technologies. Ele oferece aos usuários a capacidade de criar jogos em 3D e 2D. Sem falar na sua versatilidade e possibilidade de diversos estilos de gráficos e mecânicas para jogos.

C#

Os scripts em Unity são escritos em C# , uma linguagem de programação rica e de uso geral que é fácil de aprender. logo é necessario aprender tal linguagem para a criação de jogos.

Photoshop

Seja na criação de texturas para jogos 3D ou sprites para jogos 2D um bom editor de imagens é bastante util na criação de jogos portanto o Photoshop é uma ferramenta bastante atrativa.
